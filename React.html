What is React?

React is a JavaScript library. It is used to make a web page interactive. It was created by Facebook in 2013 and originally used for their news feed.

Instead of manipulating the browser's DOM directly, React creates a virtual DOM, where it does all the necessary manipulating. This means that the whole page doesn't need rendering, just the parts that actually changed, which makes the performance of the entire page a lot better. Learn more about React here.

React is a free and open-source library which means that anyone can use, copy and improve the software. Read more about open-source here.
